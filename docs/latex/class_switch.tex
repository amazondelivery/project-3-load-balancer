void Switch::\+tick (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})}



Advances the \doxylink{class_switch}{Switch} by one clock cycle. 

On each tick:\+
\begin{DoxyItemize}
\item Each \doxylink{class_web_server}{Web\+Server} is ticked.
\item Idle servers peek at the front \doxylink{struct_request}{Request}; if it matches this \doxylink{class_switch}{Switch}\textquotesingle{}s job type and passes the firewall, it is popped and processed.
\item If the front \doxylink{struct_request}{Request} does not match this \doxylink{class_switch}{Switch}\textquotesingle{}s type, the server stops pulling.
\item Every 10 cycles, servers are added or removed based on queue size. 
\end{DoxyItemize}
