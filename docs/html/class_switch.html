◆ consultFirewall()

bool Switch::consultFirewall ( std::string ip)

Checks whether an IP address should be blocked by the firewall.

Blocks the following ranges:

  • 192.168.x.x (private)
  • 10.x.x.x (private)
  • 172.16.x.x through 172.31.x.x (private)
  • Any IP below or equal to 20.40.60.80
Parameters
ipstd::string The IPv4 address to evaluate (e.g. "172.24.53.101").
Returns
bool true if the IP is blocked, false if it is allowed.

◆ tick()

void Switch::tick ( )

Advances the Switch by one clock cycle.

On each tick:

  • Each WebServer is ticked.
  • Idle servers peek at the front Request; if it matches this Switch's job type and passes the firewall, it is popped and processed.
  • If the front Request does not match this Switch's type, the server stops pulling.
  • Every 10 cycles, servers are added or removed based on queue size.
